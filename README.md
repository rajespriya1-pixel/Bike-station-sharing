# Personal Finance budgeting
Power BI project to analyze household income and expenses for effective budgeting and financial planning.

# Tools

Excel – Data Cleaning

Power BI – DAX & Dashboard

# Key Insights

December shows highest loss (expense > income)

Lottery & entertainment are major expense drivers

USA expense heavy; Japan & Canada income heavy

Small households spend more than mid-sized

# Outcome

Helps track spending patterns, control unnecessary expenses, and improve savings.

# Conclusion

Financial volatility is driven by non-essential spending and unstable income like lottery. Better budgeting, expense control, and focus on stable income can improve long-term financial balance.
